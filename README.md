# Predictive Analysis Using Machine Learning

## Project Overview
This project builds a machine learning model to predict diabetes outcomes based on medical data. The notebook demonstrates a complete end-to-end machine learning workflow.

## Objectives
- Data preprocessing and analysis
- Feature scaling and selection
- Model training and evaluation
- Model comparison and interpretation

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Models Used
- Logistic Regression (Accuracy: 75.32%)
- Random Forest (Accuracy: 77.27%)

## Key Features
- Feature Selection using SelectKBest
- Model Comparison
- Confusion Matrix & Classification Report
- Feature Importance Analysis

## Result
Random Forest performed better with an accuracy of 77.27%.
